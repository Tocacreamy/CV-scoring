how to commit & push:
🎉	feat :	Menambahkan fitur baru.
🐛	fix :	Memperbaiki bug.
📚	docs :	Mengupdate dokumentasi (README, tutorial, dll).
💄	style :	Perubahan gaya (indentasi, spasi, dll).
🚀	perf :	Peningkatan performa.
🔥	refactor :	Refaktor kode tanpa mengubah fungsionalitas.
🧪	test :	Menambahkan atau mengupdate pengujian.
🗑️	chore	: Perubahan kecil seperti update dependensi.
📦	build :	Perubahan terkait build tools atau deploy.

ex : 

🐛 fix: perbaiki validasi form yang error
Memperbaiki bug pada validasi input email di halaman registrasi.

🎉 feat: tambahkan fitur login dengan autentikasi JWT
Menambahkan form login dan integrasi autentikasi berbasis token JWT.
